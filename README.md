# pi-wechat-extension

微信插件，将微信接入 pi coding agent，实现"微信聊天 → AI 回复"的双向交互。

## 兼容性

| 项目 | 版本要求 |
|------|---------|
| pi | ≥ 1.0.0 |
| Node.js | ≥ 22.0.0 |

## 已实现功能

### 命令

| 命令 | 功能 | 状态 |
|------|------|------|
| `/wechat login` | 扫码登录微信 | ✅ |
| `/wechat logout` | 登出微信 | ✅ |
| `/wechat status` | 查看连接状态 | ✅ |
| `/wechat start` | 手动启动轮询 | ✅ |
| `/wechat stop` | 手动停止轮询 | ✅ |

### 核心功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 扫码登录 | ✅ | 扫码一次，登录信息持久化保存 |
| 登录状态持久化 | ✅ | 下次打开无需扫码，自动恢复连接 |
| 消息接收 | ✅ | 微信消息 → pi 会话，触发 AI 回复 |
| 长轮询 | ✅ | 实时接收微信消息 |
| /help 命令 | ✅ | 微信发送 /help 获取帮助 |

## 快速开始

### 1. 安装插件

将项目链接添加到 pi 扩展配置：
```bash
# 或通过 pi 的扩展管理界面添加
```

### 2. 登录微信

```
/wechat login
```
扫描二维码完成登录。

### 3. 开始聊天

登录后自动连接微信。直接通过微信给 bot 发消息，AI 会自动回复。
