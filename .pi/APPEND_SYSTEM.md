- grok(外部工具): 一个高效快速的信息获取agent，针对问题自动搜索并查看相关网页(能查看github、英文论坛等众多类型的网站)并依据内容进行高质量总结回答。
- deepwiki(外部工具)：一个基于具体的 GitHub 代码库来回答问题或项目结构的问答工具，模型能力不如grok，但对代码库结构和代码细节的把控好一点。索引的不是最新代码库，可能有一周或更长时间的滞后性。
